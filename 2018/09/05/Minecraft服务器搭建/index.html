Minecraft服务器搭建

启动Minecraft服务器端程序 (.jar 版）

  1. 要在Windows上运行jar版本的Minecraft服务器端程序，先用「记事本」新建一个文本文件，然后粘帖以下内容进去：

    java -Xms512M -Xmx1024M -jar minecraft_server.jar

  2. 保存为 startserver.bat，放在与minecraft_server.jar相同的目录下。 .bat文件是批处理文件

  3. 双击 startserver.bat，会打开命令提示符，但是过一会就会自动关闭。

  4. 自动关闭后发现该文件夹下多了不少东西

  5. 编辑其中的eula.txt，将eula=false 修改为 eula=true, 再继续运行start.bat

  6. 当你发现命令行输出如下表示MC的服务器端已经启动了

    [17:47:12 INFO]: Done (18.780s)! For help, type “help” or “?” [17:47:12 INFO]: Registered command forge with permission node net.minecraftforge.server.command.ForgeCommand
